## how it works

- swaps common words into degen slang (wen, dat, ur, ppl, etc)
- drops everything 2 lowercase
- kills apostrophes and unnecessary periods
- keeps da original meaning intact
- doesnt mangle it unreadable

## how 2 install

1. open claude.ai
2. go 2 **settings → profile → custom skills**
3. hit **create new skill**
4. set da name 2 `degen-translator`
5. paste da content from `SKILL.md` in dis repo
6. save it
7. done

now just tell claude "degenify this" and drop in ur text
